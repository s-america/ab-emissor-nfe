# 00 — Visão Geral do Projeto

## Projeto

**AB Emissor NF-e**

## Proprietário

Salta Digital — Sergio Figueroa `<sergio@saltadigital.com.br>`

## Cliente inicial

AB Contabilidade — Curitiba/PR — `https://www.abcont.cnt.br`

## Objetivo

Criar uma aplicação web em Laravel para emissão auxiliar de NF-e modelo 55 por clientes da AB Contabilidade, com painel para a contabilidade acessar XMLs, relatórios, rejeições, status de certificados e histórico fiscal.

## Decisão principal

O projeto será reconstruído do zero.

O sistema legado EmissorNFe será usado apenas como referência funcional, não como base de código.

## Escopo inicial

- Laravel;
- multiempresa;
- clientes da contabilidade;
- NF-e modelo 55;
- certificado A1;
- XML autorizado;
- DANFE;
- painel da contabilidade;
- auditoria;
- relatórios;
- preparação para SaaS futuro.

## Fora do escopo inicial
