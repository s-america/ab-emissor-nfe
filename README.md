# AB Emissor NF-e

Projeto Laravel para emissão auxiliar de NF-e modelo 55, desenvolvido pela Salta Digital para uso inicial da AB Contabilidade e futura evolução para SaaS comercial.

## Status

Projeto iniciado em junho de 2026.

## Decisão técnica

Este projeto é uma reconstrução do zero. O sistema legado EmissorNFe será usado apenas como referência funcional e histórica, não como base de código.

## Documentação

A documentação principal está em `docs/arquitetura-ab-emissor-nfe-v1.0.md`.

## Ambiente inicial

- Desenvolvimento: Windows 10
- IDE: Visual Studio Code + Codex
- PHP local atual: XAMPP PHP 8.2.12
- Banco local: MariaDB 10.4.32
- Framework: Laravel
- GitHub: `s-america/ab-emissor-nfe`
- Deploy futuro provável: subdomínio da AB Contabilidade na Locaweb
